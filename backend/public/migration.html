Migration Details

This migration will add support for a second button in the hero banner:

  • Button2Text - Text for the second button (default: "Custom Design")
  • Button2Link - Link for the second button (default: "/custom-furniture")
  • Button2BgColor - Background color for the second button (default: "#6c757d")
  • Button2TextColor - Text color for the second button (default: "#ffffff")

Note: This migration is safe to run multiple times. It will only add columns that don't already exist.
